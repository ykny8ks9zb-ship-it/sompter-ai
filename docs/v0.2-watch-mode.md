# Sompter v0.2 — Always-On Mac Agent

## Vision

An AI that can see your screen at all times, understand what you're doing,
use the web, and talk to you through Apple Notes.

## Architecture

### 1. Always-on Screen Watcher

- Screenshot every 3–5 seconds (not continuous recording)
- Uses Electron `desktopCapturer` / `screencapture -x`
- Downscaled/compressed before sending to backend
- Only latest screen state stored (no logging unless enabled)
- Requires: System Settings → Privacy → Screen & System Audio Recording

### 2. Apple Notes Bridge

- One shared note named "Sompter Chat"
- Assistant writes replies into the note
- Reads user lines starting with "You:" from the note
- Uses `osascript` / AppleScript via Node.js

### 3. Web Agent (Phase 2)

- Playwright browser automation (already integrated)
- Open pages, click, type, fill forms, scrape
- Safety gates: ask before logins, purchases, messages, passwords

### 4. Mac Control (Phase 3)

- Click/type via PyAutoGUI (already integrated)
- App opening, file control, terminal commands
- Safety confirmations + Emergency Stop

## Safety Rules

- No autonomous purchases, messages, passwords, deletions
- No terminal commands, account changes, or betting actions
- Risky actions return `needs_confirmation: true`
- Always ask before: logins, purchases, messages, emails, password entry
- Emergency Stop button in UI and tray menu
